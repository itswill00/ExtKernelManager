# Ext Kernel Manager

A simple and efficient kernel manager for Android, built with Jetpack Compose. This app is designed to give you clean, direct control over your hardware without unnecessary complexity.

## Features

- **Real-time Dashboard**: Monitor CPU, GPU, RAM, and storage usage at a glance.
- **Hardware Control**: Easily adjust CPU/GPU frequencies and governors.
- **System Info**: View detailed kernel version, build info, uptime, and deep sleep status.
- **Memory Management**: Manage ZRAM and Low Memory Killer (LMK) profiles.
- **Clean UI**: A modern interface following Material 3 guidelines.

## Requirements

- A rooted Android device (Magisk or KernelSU).
- Android 11 or higher.

## How to Build

Import the project into Android Studio and run:
```bash
./gradlew assembleDebug
```
